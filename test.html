SmartSaver Connect — Setup Test

Use this page to verify your Apps Script URL is working before updating index.html

Step 1 — Get your Web App URL

In your Google Sheet: Extensions → Apps Script → Deploy → Manage deployments

Copy the Web App URL (looks like: https://script.google.com/macros/s/AKfy.../exec)

Step 2 — Test the connection

Step 3 — Update your files

Once the test passes, paste the same URL into:

  • index.html — find YOUR_APPS_SCRIPT_WEB_APP_URL_HERE and replace it
  • compare.html — same replacement

Then push both files to GitHub.

Still not working?

  • Make sure the Web App is deployed with Execute as: Me and Who has access: Anyone
  • After any code change, create a New deployment (not just save)
  • Run at least one scraper first: Google Sheet → SmartSaver Connect menu → Test scrapers
